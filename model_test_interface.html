猜测熵（GE）

衡量键盘被声学攻击的难度，GE值越高，安全性越强。基于融合模型的口令正确率计算。

🎯

融合模型技术

结合声音模型和口令模型的双重分析，大幅提升攻击准确率，更真实地反映安全风险。

📊

安全等级划分

根据GE值将键盘分为五个安全等级：A级（极高）、B级（高）、C级（中等）、D级（低）、E级（极低），便于快速识别风险。

评分计算公式

第一步：计算猜测熵
GE = -log₂(P)
P = 融合口令正确率（如95% = 0.95）
第二步：归一化得分
S = 100 × (GE - GE_min) / (GE_max - GE_min)
GE_min = 0.07，GE_max = 4.32
计算示例

以戴尔笔记本为例：融合口令正确率25%

GE = -log₂(0.25) = 2.00

得分 = 100 × (2.00-0.07)/(4.32-0.07) = 45.4分 → C级

安全使用建议

🏢

企业办公环境

建议选择GE值≥2.0的键盘，避免在开放办公区使用极低安全等级的键盘输入敏感信息。

🏠

个人日常使用

中等安全级别即可满足需求，但在输入密码时应注意周围环境的隐私性。

🔒

高安全场景

处理机密信息时，应选择高安全等级键盘，并配合其他安全措施如屏幕键盘。

系统概述

音落留痕（ASCP）是一个专注于键盘声学侧信道安全的研究平台。系统基于深度学习技术， 通过分析键盘敲击声音特征，结合先进的口令猜测模型，展示声学侧信道的潜在安全风险， 为输入设备安全防护技术的研究与发展提供支持。
